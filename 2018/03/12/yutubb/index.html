Web Dev Trends 2018

Trend # 1: Pahami Dasar-dasarnya

Menjadi hal terpenting yang harus Anda ketahui sebagai pengembang web pada tahun 2018!
kita memiliki lebih banyak teknologi untuk pengembangan web, penting untuk kita memahami apa yang ada di dalamnya.

Trend # 2: Menyelam ke Node.js

Node.js adalah bahasa yang menawarkan kinerja hebat. Ini menampilkan ekosistem yang dinamis dan aktif dengan banyak paket pihak ketiga yang dapat Anda pasang ke aplikasi Anda.

Trend # 3: Jelajahi React, Angular dan Vue.js (Kerangka Pengarah)

Framework frontend yang populer di tahun 2018 adalahAngular, React dan Vue.js.
Kerangka kerja ini memungkinkan pengguna mengontrol di aplikasi web melalui JavaScript. Sehingga dapat membuat ulang bagian halaman (atau keseluruhan halaman) setelah dimuat. Ini menghemat pemuatan ulang halaman yang tidak perlu dan meminta siklus tanggapan <-> dengan server.

Trend # 4: Komponen Web & Komponen Kompiler

Sampai akhir 2017, dukungan browser adalah masalah yang lebih besar namun pada awal 2018, ini terlihat lumayan. Empat API Web yang membentuk komponen web - Unsur Khusus, DOM Shadow, Template HTML dan Impor HTML - didukung oleh semua browser utama.

Dengan elemen khusus dapat membuat elemen HTML Anda sendiri (misalnya ). Unsur-unsur ini berisi semua kode yang dibutuhkan untuk membawa sesuatu ke layar.

Trend # 5: Gunakan Generator Halaman Statis

Generator Halaman Statis seperti Jekyll, Hugo atau proyek seperti Nuxt.js untuk Vue.js (ini terinspirasi oleh Next.js for React) menyajikan versi halaman web yang telah ditentukan sebelumnya tanpa secara dinamis melakukan pra-render di beberapa server.

Trend # 6: Aplikasi Web Progresif (PWA)

Progressive Web Apps (PWA) adalah teknologi yang memungkinka kami membuat aplikasi web yang berperilaku sangat mirip dengan aplikasi seluler asli.
File manifest.json di sisi lain memberikan informasi meta yang berguna tentang aplikasi Anda ke browser, yang memungkinkan aplikasi dipasang di halaman depan.

Aplikasi bisa dipakai offline (misalnya, bagian dari aplikasi web yang berfungsi tanpa sambungan aktif - setelah aplikasi web dimuat setidaknya satu kali), kemampuan terinstal (misalnya, aplikasi web dapat ditambahkan ke layar utama perangkat) dan penggunaan perangkat asli fitur (seperti mengakses kamera perangkat).

Trend # 7: Bangunlah Aplikasi Tanpa Server

Serverless dapat dikatakan minim dalam mengelola dan memperbarui server, sehingga kapasitas server meningkat drastis dan Anda tetap hanya membayar untuk apa yang Anda butuhkan.
Cara bekerjanya adalah :

  • menulis kode Anda yang harus dijalankan sesuai permintaan
  • mengunggahnya ke layanan tanpa server(misalnya AWS Lambda)
  • menambahkan beberapa layanan tambahan (misalnya, AWS API Gateway untuk membangun REST tanpa server API)
  • menerapkan aset statis ke beberapa host statis seperti AWS S3.

Trend #8: Artificial Intelligence dan Bots

AI TIDAK terbatas pada mobil self-driving, hat bots di laman membantu pengguna memahami konten, alat pengenalan gambar dan pengoptimalan.
Ada banyak use-cases untuk komponen AI-driven di halaman web. Belajarlah Python, Machine Learning, Big Data, Deep Learning dan membangun alat AI sendiri.

Dan kita akan melihat banyak alat seperti ini muncul (dan menjadi lebih baik) di tahun 2018.
